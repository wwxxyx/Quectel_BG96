Accessories

HomeKit Accessory Simulator contains one Bluetooth LE accessory, but can have multiple IP accessories (including regular IP accessories, bridges, bridged accessories, IP cameras, and video doorbells) added. The Bluetooth LE accessory can not be deleted or exported.

To create an IP accessory in HomeKit Accessory Simulator, select the "+" located in the lower-left corner of the app and select "New Accessory…"; use the keyboard shortcut ⌘N; or from the File Menu, select "Accessory…" under the "New" menu item. Enter the accessory information when prompted and select the "Finish" button.

To create a bridge accessory in HomeKit Accessory Simulator, use the keyboard shortcut ⌘B; or from the File Menu. Alternatively, an existing IP accessory can become a bridge accessory by adding a bridged accessory to it as described below.

To add a bridged accessory to a bridge, first select an IP accessory from the accessory list on the left side of the window. Then select the "+" located in the lower-left corner of the app and select "New Bridged Accessory…"; use the keyboard shortcut ⇧⌘B; or from the File Menu, select "Bridged Accessory…" under the "New" menu item. Enter the bridged accessory information when prompted and select the "Finish" button.

To create an IP camera in HomeKit Accessory Simulator, simply creata a new IP accessory and select IP Camera from the list of accessory types of the presented accessory wizard window. Alternatively, you can use the keyboard shortcut ⇧⌘C; or from the File Menu, select "IP Camera…" under the "New" menu item.

To create a video doorbell in HomeKit Accessory Simulator, simply creata a new IP accessory and select Video Doorbell from the list of accessory types of the presented accessory wizard window. Alternatively, you can use the keyboard shortcut ⌘D; or from the File Menu, select "IP Camera…" under the "New" menu item.

Accessories may be deleted by selecting them from the accessory list, and then either pressing the delete key or selecting "Delete" from the "Edit" menu. They can also be deleted by selecting them and clicking "-" in lower-right corner of accessory list. A Bluetooth LE accessory can't be deleted.

Please note that the Bluetooth LE accessory does not support becoming a bridge.

Names must be unique. If a newly created accessory, bridge, or bridged accessory name is already being used, it will automatically be renamed.
